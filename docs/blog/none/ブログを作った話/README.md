# Blogを作った話

Github pagesを使ってブログを作成した。

今回は、Hugoを使ったりしておしゃれなページは作っていない。

やる気がないブログではあるものの、ごちゃ付いていないので個人的には好き。

## こだわりポイント

**こだわりポイント その１**

Blogのasciiアートはアクセスごとにランダム。

なのでリロードしてもらえると多少は楽しめるはず。

**こだわりポイント その２**

良いか悪いかはおいておいて、ブログのタイトルを押すと遷移はせず内容が表示される。

本当はGithub Actionsでmdファイルをhtmlに変換してといったことを考えていたが、めんどくさいのでやめた。

ただ、fetchしてtextContentでpreに書くだけ。
